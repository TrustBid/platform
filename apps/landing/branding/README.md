# Branding assets

Iconos de marca que **no** se usan en el sitio web (no se empaquetan en el build).

- `app-icon-1024.png` — Icono estilo app (tile azul `#0B28FE` + símbolo blanco, esquinas
  redondeadas). 1024×1024, pensado para el icono de la **futura app** (App Store / Play Store).
- `app-icon-512.png` — Misma versión a 512×512.

> El favicon del sitio (pestaña del navegador) usa el símbolo azul transparente,
> en `public/favicon-*.png` y `public/apple-touch-icon.png`.
